# `hoursToMinutes()`
Returns the number of minutes in the specified number of hours.

### Usage
```javascript
import { hoursToMinutes } from 'easy-dates';

hoursToMinutes(hours)
```

### Prop types

| prop    | type     | required | default |
|---------|----------|----------|---------|
| `hours` | `number` | yes      | n/a     |

### Examples
```javascript
hoursToMinutes(1) // 60
```

```javascript
hoursToMinutes(2) // 120
```
