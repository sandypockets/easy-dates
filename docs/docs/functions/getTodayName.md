# `getTodayName()`
Returns the name of today's day.

## Usage
```javascript
import { getTodayName } from 'easy-dates';

getTodayName() // Friday
```

## Prop types
This function does not take in any props.

| prop | type | required | default |
|------|------|----------|---------|
| n/a  | n/a  | n/a      | n/a     |
